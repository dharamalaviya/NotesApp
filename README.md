## NotesApp
#### Create the simple NotesApp using File Manager
### Implemented Elements
* UILabel
* UITextView
* UITextField
* UIButton
* UIImageView
* UIBarButtonItem
* UITableView
* UIAlert
* UserDefault
* FileManager
### Following task performed in application
* Login
* Logout
* Add new Notes
* Update existing Notes
* Delete Notes
